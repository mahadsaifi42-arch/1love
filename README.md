# Ultra Discord Bot

### Setup
1. **Discord Portal:** Sabhi 3 Intents (Presence, Members, Message Content) on karein.
2. **Render:**
   - Build Command: `npm install`
   - Start Command: `npm start`
   - Env Variables: `DISCORD_TOKEN`, `OWNER_ID`, `PREFIX`, `PORT`.

### Features
- **Whitelist Categories:** Ban, Mute, Prefixless, Lock, Purge, Hide.
- **Prefixless:** Whitelisted users direct `lock` ya `ban` likh sakte hain.
- **Music:** Stable `play-dl` system.
- **AFK:** Both prefix and prefixless support.
- **Style:** Pure black embeds with premium emojis.
